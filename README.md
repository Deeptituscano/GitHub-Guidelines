# GitHub Guidelines

## Repository Name
1. Should be in Title Case.
2. Title should be Hyphenated.
3. Please don't use Colon or Underscore.

#### Example:
| Incorrect | Correct |
----------------------
|


## Chapter title in Readme file
1. Should be in Title Case
2. Title should not be hyphenated
3. 
| Correct | Incorrect|
----------------------
|
